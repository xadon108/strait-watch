## What It Shows

- **KPI Cards**: vessels in transit, queued/waiting count, average wait time, 24-hour throughput
- **Interactive Map**: vessel positions on a dark-themed Leaflet map
- **24-Hour Transit Chart**: hourly transit volume (Chart.js)
- **Vessel Register**: table with name, IMO, flag, type, status, speed, heading, last-seen
- **Countdown Timer**: live countdown to the 6 April 2026 deadline

## Tech Stack

Leaflet.js, Chart.js, Vanilla JS, GitHub Pages

## Quick Start

1. Clone or download this repo
2. Open `index.html` in any browser
3. Dashboard runs immediately in simulation mode
